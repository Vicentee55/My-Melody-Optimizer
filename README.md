# My Melody Optimizer

Ferramenta de otimização para Windows focada em **desempenho, FPS e redução de latência** em jogos.

Aplica ajustes diretos no sistema como Windows Update, Game DVR, BCDEdit e kernel gráfico.

---

## Como usar

1. Clique com o botão direito no arquivo `.bat`  
2. **Executar como administrador**  
3. Pressione **1** para aplicar  
4. Reinicie o PC  

---

## Aviso

- Crie ponto de restauração  
- Não use com conta Microsoft ou PIN  
- Uso por sua conta e risco
